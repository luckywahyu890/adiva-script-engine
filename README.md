# ⬡ ADIVA Script Engine

Lightweight 2D WebGL Game Engine for Browser Game Development.

ADIVA Script Engine is a simple and fast game engine designed for creating 2D browser games using WebGL.  
It focuses on **easy workflow, separated image assets, physics system, and rapid prototyping**.

---

## 🚀 Features

- ⚡ Lightweight WebGL-based renderer
- 🎮 Simple game loop system
- 🧠 Object-based entity structure
- 🧲 Built-in collision (AABB)
- 🏃 Platformer physics (gravity, jump, double jump)
- 🎥 Camera follow system
- 🎞️ Frame-based animation system
- 🧩 Scene / game state system
- 📱 Mobile virtual buttons support
- 🔊 Audio system for sound effects
